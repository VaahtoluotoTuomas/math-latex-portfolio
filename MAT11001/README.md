# MAT11001: Introduction to University Mathematics

This repository contains my coursework, weekly exercises, and exam papers for the course **MAT11001: Introduction to University Mathematics** (Johdatus yliopistomatematiikkaan) completed at the University of Helsinki. You can read more about the course here: https://studies.helsinki.fi/kurssit/opintojakso/otm-e034179c-f679-419f-9bcc-671b20dd92aa/MAT11001

## About the Course
The course acts as a introduction to higher-level mathematics. Key topics covered include:
* Propositional and predicate logic
* Set theory, relations, and functions
* Mathematical proof techniques (direct proof, proof by contradiction, and mathematical induction)
* Cardinality and real numbers
* Complex numbers

## Technical Details
All documents in this folder have been typeset using **LaTeX** to ensure clean, professional, and standard-compliant mathematical notation. The custom formatting and macros used across these documents are centralized in the included `JYM-harj.sty` style file.
